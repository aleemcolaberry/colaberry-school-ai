# Storyboard & Shot List — Concept 5: "Watch an idea become an AI system in 60 seconds"

**Source concept:** [viral-video-concepts.md](viral-video-concepts.md) → Concept 5
**Format:** Live build / screen-capture demo · **Platform:** YouTube Shorts + TikTok (9:16) · **Length:** 60s
**Goal:** Tangible "it actually works" proof → drive Open House registration.
**Talent:** 1 presenter (voiceover) · **Capture:** screen recording + face cam (optional PiP)

---

## Pre-production checklist
- [ ] Rehearse the build so it completes **inside 50s** (leave buffer for hook + CTA).
- [ ] Pre-stage the prompt, Skill definition, and a clean desktop (hide bookmarks/notifications).
- [ ] Brand kit ready: Cherry Red `#`, Leaf Green, Berry Blue overlays + Quicksand/Roboto fonts.
- [ ] Lower-third + countdown timer asset (top-right).
- [ ] Confirm CTA URL (`learn.colaberry.com` vs. dedicated Open House link).
- [ ] Captions burned in (assume muted viewing).

## Editing notes
- **Pacing:** hard cuts, speed-ramps over loading/dead time. No clip longer than ~6s.
- **Captions:** every VO line on-screen; key phrases in brand color pops.
- **Sound:** subtle ticking under the countdown for tension; soft "success" sting on shot 5.
- **Loop seam:** end frame can cut back to the hook for seamless replay (boosts watch-time).
- **Thumbnail/cover:** the countdown `0:60` + "sentence → agent" — curiosity gap.

## Variants to A/B (cheap, same footage)
- **Hook swap:** "60 seconds" vs. "before this timer hits zero."
- **CTA swap:** "Link in bio" vs. "Comment 'BUILD' and I'll send it."
- **Faceless cut:** screen-only version for TikTok algorithm testing.

## Distribution
- Post native to TikTok + YT Shorts + Reels (no watermark cross-post).
- Pin a comment with the registration link.
- Repurpose shot 4–5 (the "it works" beat) as a 6s teaser/ad hook.
